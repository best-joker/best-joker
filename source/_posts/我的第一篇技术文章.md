## 新帖子

每次增加新的帖子时，通过指令

```shell
hexo new "新帖子标题"
```
或者手动在source/_posts中手动创建一个.md文件和相应的文件夹

编辑新的帖子的md文件。

编辑之后再运行

```she
hexo s # 或hexo server(s的全写)
```

再产生新的文件

```she
hexo g # 或hexo generate
```

并部署

```she
hexo d # 或hexo deploy
```

## 新帖子注意事项

开头加上分类和标签实现管理

```
---
title: Hello World
date: 2024-01-01 12:00:00
categories: 
  - 博客搭建
  - Hexo教程
tags:
  - Hexo
  - 博客
  - 入门指南
  - 静态网站  
photos:
  - /images/小丑.jpg
---
```

应因为加上了`title`属性，所以原始的md文件可以不加一级标题。

以上属性均为可选。
